List of Observables¶

The following is a (incomplete) list observables that EOS can predict.

Observables in (semi)leptonic \(b\)-hadron decays¶

Observables in \(B^-\to \ell^-\bar\nu\) decays¶

Qualified Name

Description

B_u->lnu::BR

\(\mathcal{B}(B^- \to \ell^-\bar\nu)\)

The option “l” selects the charged lepton flavour.

Qualified Name

Description

\(\mathcal{B}(B\to\pi\ell^-\bar\nu)\)

B->pilnu::dBR/dq2

\(d\mathcal{B}(B\to\pi\ell^-\bar\nu)/dq^2\)

The option “l” selects the charged lepton flavour. The option “q” selects the spectator quark flavour. The option “form-factors” selects the form factor parametrization.
